# **Dynamic Programming**

![Image]({{ site.baseurl }}/assets/images/root/dynamic-programming.png){: width="1000" .d-block .mx-auto }

{: .note }

> _If you can recurse it, you can iterate it._
> _And if you can iterate it, you can recurse it._

Dynamic programming is basically rewriting the above statement as:

{: .note }

> _If you can recurse it, you can iterate it and vice versa_.
